# HRV + Cortisol Based Stress Detection System

## Overview

This project presents a real-time intelligent stress monitoring system that combines Heart Rate Variability (HRV) analysis with cortisol measurements and physiological signals.

The system integrates wearable sensors, embedded hardware, signal processing, machine learning, and a real-time monitoring dashboard for continuous stress assessment.

---

## Features

- ECG & PPG Signal Acquisition
- HRV Feature Extraction
- Cortisol Level Integration
- Machine Learning Based Stress Classification
- ESP32 Based Hardware
- OLED Display
- Real-Time Dashboard
- Cloud Ready Architecture
